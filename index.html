How to install

💻 BlueStacks (PC)
📱 Android phone
  1. Click Download APK above and save Prompted-beta.apk somewhere easy, like your Desktop.
  2. Open BlueStacks.
  3. Drag the Prompted-beta.apk file straight onto the BlueStacks window — it installs automatically.
    (Or use the Install APK button in the BlueStacks side toolbar and pick the file.)
  4. Once it finishes, find the Prompted icon on the BlueStacks home screen and open it.
  1. On your phone, open this page and tap Download APK.
  2. When prompted, allow your browser to install unknown apps (Android will walk you through this the first time).
  3. Open the downloaded Prompted-beta.apk from your notifications or Files app and tap Install.
  4. Open Prompted from your app drawer.
"Unknown sources" / Play Protect warning is expected. This is a beta build installed outside the Play Store, so Android (and BlueStacks) will warn you before installing. That's normal for testing — tap Install anyway / More info → Install.

Good to know

  • Sign in with Google, GitHub, or email — same account as the website.
  • Swipe right to open the tabs sidebar, swipe left for notifications, daily reward, and the daily Builds / Discussions / Questions.
  • Pro is purchased on the website (prmpted.com), not inside the app. Your Pro status carries over once you're signed in.
  • This is a debug/beta build — expect rough edges. Please send any bugs or feedback back to me.
Updates are manual during beta: when a new version drops, download the new APK from this page and install it over the old one (your data stays in your account).
